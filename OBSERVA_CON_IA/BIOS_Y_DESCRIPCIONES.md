## Convenciones de copy (para mantener consistencia)

- Siempre usar el verbo **"observar"** como gancho conceptual del canal
  (conecta con el nombre y el ángulo unificador).
- En piezas de pilar político, el copy nunca debe sugerir una postura —
  usar verbos como "analizar", "detectar", "comparar", nunca "denunciar"
  o "criticar".
- Hashtags base sugeridos: #InteligenciaArtificial #MarcaPersonal
  #AnálisisPolítico #IAaplicada — sumar 2-3 específicos del tema de cada
  video.
- CTA por plataforma: YouTube → "suscribite"; Instagram → "guardá este
  post"; TikTok → "seguime para la parte 2".
